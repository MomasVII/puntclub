Note: Please refer to usage.php for usage information, this text file offers general and conceptual information.

This library is designed to manage CSV data and documents. It has been heavily extended in an effort to allow for SQL 
style usage.

It can not only read and write CSV data, but can modify existing data, sort data, read and write column headers,
auto discover delimiters and save/output files.
