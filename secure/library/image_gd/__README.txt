Note: Please refer to usage.php for usage information, this text file offers general and conceptual information.

This library is designed to manage images that are already on the server. It can do a series of rudimentary image
management tasks. At some point this library will be superseded by and ImageMagick library.

At the time of writing, Scion CMS uses this library for all of it's image tasks.
